Mental Health Tracker
A colorful, interactive web app that helps users check in with their mood and receive a positive affirmation.
This project is a personal wellness tool built with HTML, CSS, and JavaScript, featuring a vibrant UI, animated effects, and a custom affirmation wheel. It encourages users to pause, reflect, and receive a supportive message based on their mood.
